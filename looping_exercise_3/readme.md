`Challenge`

```
1. Prompt the user for an integer.
2. Create an array with a number of elements equal to the user input.
3. Loop through the array. On each iteration prompt the user for an integer and add it to the array.
4. While in the loop also display which iteration the loop is on and the user input.
5. Display the highest value in the array.
```
