`Challenge 1`

```
1. Set up a collection that can be used like a deck. I.e you can add cards to it, remove cards from it, and shuffle it.
2. Create a Card class. It should have 2 properties, suit and value.
3. A standard playing card deck has 52 cards, 4 suits, and 13 cards per suit.
4. Loop through your collection and prompt the user to create each card.
5. Inside that loop display each card as the user creates them.
6. At the end of the loop  Add the card to the collection.

```
