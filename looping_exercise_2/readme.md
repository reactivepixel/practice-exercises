`Challenge`

```
1. Create an array with at least 3 elements.
2. Give a value to each index.
3. Find the index that has the highest length.
4. Loop through the array. You will need to add spaces to each element so each element has the same length.
5. Add your frame character to each element.
6. Wrap your array in a frame.

```

`Example`

```
*******
*Hello*
*World*
*!    *
*******
```
